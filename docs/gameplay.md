# 玩法说明

## 基本流程

1. 玩家 A 先手，首回合选择 1 名武将。
2. 玩家 B 后手，第二回合选择 2 名武将。
3. 后续双方轮流，每回合可以选择 1 到 2 名武将。
4. 双方 8 个位置都补满后，进入阵容调整与 AI 裁决阶段。

## 位置设计

| 位置 | 核心倾向 |
| --- | --- |
| 主公 | 只允许蓝色英雄，统率权重最高 |
| 军师 | 智力权重最高 |
| 元帅 | 武力、智力、统率较均衡 |
| 先锋 | 武力权重最高 |
| 骑兵 | 武力与机动型角色更强 |
| 水军 | 智力与统率都有价值 |
| 后勤 | 统率与续航型角色更稳 |
| 间谍 | 智力与潜伏能力更占优 |

## 技能卡

- 消卡：移除双方同一位置的武将
- 换卡：交换双方同一位置的武将
- 禁人：禁用一个还未上场的武将
- 任选四：跳过点将，直接从剩余武将里选择
- 重摇：对当前候选武将重新抽取一次

## AI 裁决

AI 会按位置权重计算每名武将的基础战力，再叠加少量随机波动：

- 差距很小时判定为平局
- 差距明显时判定出该位置胜负
- 全部位置结算后统计总分并选出 MVP
